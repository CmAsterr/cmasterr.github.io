# 🧮 跨平台一元多项式可视化计算器 (PolyCalc Web)

✨ **本项目为《数据结构》课程设计的 Web 端实现**。项目采用了现代化的前后端分离架构与纯前端渲染技术，并且原生支持 **PWA (渐进式 Web 应用)**。无论您使用的是 Windows、macOS 还是 Android / iOS 移动端，均可直接将其添加到桌面，作为独立的 App 离线运行。

👉 **立即在线体验：[cmasterr.github.io](https://cmasterr.github.io)**

---

## 🌟 核心特性

本项目打破了传统数据结构课程设计仅限“控制台黑框”的局限，在完成基础要求的前提下，横向扩展了大量高级代数与可视化功能：

### 1. 基础多项式代数 (课程设计核心要求)

* **多项式运算**：支持一元多项式的加法、减法和乘法。
* **双存储结构支持**：底层同时实现了**顺序存储**（基于连续数组）与**动态存储**（基于 ES6 Map 哈希映射），并提供内存开销实时估算与对比。
* **规范化处理**：运算结果自动落实“合并同类项”与“销毁零系数项”的底层机制。
* **智能稀疏判定**：根据最高次幂与实际项数的比值，自动判定多项式的稀疏/稠密属性。

### 2. 高级代数矩阵 (创新扩展功能)

* **AST 表达式化简**：内置词法分析器与抽象语法树（AST）解析器，支持复杂嵌套括号的展开与常数折叠化简。
* **一元方程求解**：内置复数运算体系，支持最高四次一元方程的解析解（涵盖一次/二次公式、卡尔丹公式与费拉里法）。
* **多域因式分解**：支持在整数域、有理数域及无理数/复数域进行多项式的因式分解提取。
* **多项式求值**：支持变量的高精度代入求值。

### 3. 数学可视化与交互引擎

* **LaTeX 实时排版**：引入 KaTeX 引擎，将所有抽象的底层数据结构实时转化为标准的数学公式。
* **二维函数图像重绘 (Canvas)**：
  * 支持建立逻辑“世界坐标系”并自动映射至屏幕物理像素。
  * **深度适配移动端手势**：完美支持单指拖拽平移、双指捏合缩放（Pinch-to-Zoom，以双指中心为锚点），以及单指点击曲线精准吸附坐标。
* **Glassmorphism 拟物化 UI**：全站采用现代化的 iOS 液态玻璃风格设计，支持全局“白天 / 黑夜”模式的无缝切换与 3D 转盘路由动画。

## 📁 项目架构

本项目采用纯前端技术栈（HTML5 / CSS3 / Vanilla JavaScript），零构建工具依赖，开箱即用：

```text
cmasterr.github.io/
├── index.html                # PWA 应用主入口重定向
├── manifest.json             # PWA 应用清单（定义应用名称、图标等）
├── sw.js                     # Service Worker 核心脚本（离线缓存控制）
├── html/                     # 核心页面模块（各独立功能的视图层）
│   ├── index.html            # 表达式化简页
│   ├── operate.html          # 多项式运算页
│   ├── plot.html             # 函数图像绘制页
│   └── ... (其他功能页)
├── css/                  
│   └── app.css               # 全局样式表（控制液态玻璃 UI 与响应式布局）
├── js/                       # 核心逻辑驱动层
│   ├── poly_core.js          # 核心算法引擎（AST解析、顺序/动态存储多项式运算核心库）
│   ├── ui_shared.js          # 全局公共 UI 控制
│   ├── tabs_scroll.js        # 导航栏逻辑与昼夜模式切换引擎
│   ├── page_transition.js    # 页面间 3D 转场动画引擎
│   └── page_*.js             # 各功能页对应的独立业务逻辑控制器
└── latex/                    # 外部依赖库 (KaTeX 等)
```

## 🚀 如何运行与安装

### 方案一：在线访问 (推荐)

直接通过现代浏览器访问 [cmasterr.github.io](https://cmasterr.github.io)。

### 方案二：PWA 桌面/主屏幕安装

* **PC 端 (Chrome/Edge)**：访问上述网址后，点击浏览器地址栏右侧的“安装 PolyCalc”图标。
* **iOS 端 (Safari)**：访问上述网址，点击底部菜单的“分享”按钮，选择“添加到主屏幕”。
* **Android 端**：访问上述网址，在弹出的提示框或浏览器菜单中选择“安装应用”或“添加到主屏幕”。

### 方案三：本地克隆运行

本项目为纯静态文件，无需配置 Node.js 或数据库环境。

1. 克隆本项目：
   `git clone https://github.com/CmAsterr/cmasterr.github.io.git`
2. 为了避免浏览器的跨域限制（CORS）导致本地图片或字体无法加载，建议使用轻量级本地服务器运行（如 VS Code 的 Live Server 插件）。
3. 使用 Live Server 打开 `index.html` 即可运行。
